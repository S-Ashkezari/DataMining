Learning Pandas from Scratch#

  1. Introduction to Pandas What is Pandas? Pandas is a powerful open-source data manipulation and analysis library for Python. It provides data structures and functions needed to efficiently work with structured data. It is built on top of NumPy, which provides support for large, multi-dimensional arrays and matrices. Pandas is widely used in data analysis, data science, and machine learning.

Why Use Pandas for Data Analysis? Easy handling of missing data. Efficient data slicing, filtering, and grouping operations. Powerful tools for data cleaning and wrangling. Support for importing and exporting data from various formats like CSV, Excel, SQL databases, and more. Built-in functions for quick data visualization.

2. Data Structures in Pandas#

Pandas provides two primary data structures:

Series: One-dimensional labeled array, similar to a list but with labels (index). DataFrame: Two-dimensional labeled data structure, similar to a table or spreadsheet.

2.1 Series#

A Series is a one-dimensional labeled array capable of holding any data type (integers, strings, floats, etc.). It can be thought of as a column in a spreadsheet.

Creating a Series You can create a Series from a list, NumPy array, dictionary, or even a scalar value. Here are a few examples:

import pandas as pd

# Creating a Series from a list
data = [10, 20, 30, 40, 50]
series = pd.Series(data)
print(series)

# Creating a Series with custom index
series_custom_index = pd.Series(data, index=['a', 'b', 'c', 'd', 'e'])
print(series_custom_index)

Series Operations#

Series support vectorized operations similar to NumPy arrays:

# Arithmetic operations
print("\nAdding 5 to each element:")
print(series + 5)

# Conditional filtering
print("\nFiltering elements greater than 20:")
print(series[series > 20])

Accessing Series Elements#

You can access elements using both integer-location-based indexing and label-based indexing:

# Accessing by position
print("\nElement at position 2:")
print(series[2])

# Accessing by label
print("\nElement with label 'c':")
print(series_custom_index['c'])

2.2 DataFrame#

A DataFrame is a two-dimensional labeled data structure with columns of potentially different data types. It can be thought of as a table or spreadsheet.

Creating a DataFrame#

You can create a DataFrame from a dictionary of lists, a list of dictionaries, or even NumPy arrays.

# Creating DataFrame from a dictionary
data = {
    'Name': ['Alice', 'Bob', 'Charlie', 'David'],
    'Age': [25, 30, 35, 40],
    'Salary': [50000, 60000, 70000, 80000]
}
df = pd.DataFrame(data)
print(df)
print("*"*50)
# Creating DataFrame from a list of dictionaries
data = [
    {'Name': 'Alice', 'Age': 25, 'Salary': 50000},
    {'Name': 'Bob', 'Age': 30, 'Salary': 60000}
]
df = pd.DataFrame(data)
print(df)
